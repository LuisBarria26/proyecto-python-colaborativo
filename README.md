# Proyecto Python Colaborativo

## Descripción
Proyecto desarrollado en Python que contiene módulos independientes para realizar operaciones matemáticas e imprimir un cuento.

## Módulos

### operaciones.py
- Suma
- Resta
- Multiplicación
- División

### cuento.py
Imprime un cuento corto en pantalla.

### main.py
Ejecuta las operaciones matemáticas y muestra el cuento.

## Ejecución

python main.py
